# Notas del proyecto Pure Data

Carpeta para guardar aprendizajes, referencias y snippets útiles mientras trabajamos con Pd.

## Estructura sugerida
- `midi.md` — configuración MIDI, dispositivos, trucos
- `audio.md` — síntesis, objetos de audio útiles
- `comunicacion.md` — pdsend/pdreceive, OSC, puertos
- `patches/` — patches de referencia comentados
- `snippets.md` — fragmentos de código Pd o Python reutilizables
